# orderbook

Experimental orderbook approach. Inspired by [c1982/orderbook](https://github.com/c1982/orderbook) for Crystal Language

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  orderbook:
    github: umutuluer/orderbook
```

and then

```
shards install
```
